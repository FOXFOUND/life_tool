# 相亲对象评估打分表
## 一、评分体系总规则
### 设计依据
完全锚定断层线全链路分析7层框架：通过标准化提问完成**输入信息净化**，以大众普遍认知为**自然基准校准**，从回答中拆解思维闭环、原生家庭结构、人性底层特质，最终量化识别三观、逻辑、性格、边界层面的核心断层点。

### 权重逻辑
按对长期婚恋关系的影响深度排序权重，核心层（婚恋三观、原生家庭、性格特质）占比66%，次核心层（处事逻辑、认知能力、消费习惯）占比34%，总分100分。

### 评分档位规则
每题设3个评分档位，对应不同系数，单题得分=单题分值×档位系数：
- 优秀（系数1.0）：观点辩证不极端，有具体事例支撑，权责认知对等，具备反思意识，逻辑闭环完整
- 合格（系数0.6）：观点符合大众普遍认知，无明显极端倾向，逻辑无硬伤但深度不足，无核心矛盾
- 不合格（系数0.2）：观点极端片面，单一归因，存在明显双标，逻辑断裂，被网络叙事严重裹挟

### 一票否决红线（核心断层触发规则）
以下8项为系统临界崩溃点，**无论总分高低，触发任意1项即判定为高风险，建议直接终止接触**：
1. 婚恋权责极端双标：明确主张“男性必须承担全部经济责任，女性无需承担家务/育儿义务”等权责完全不对等的观点
2. 原生家庭边界完全缺失：明确表示“人生所有决策都听父母安排，婚姻完全由父母决定”，无独立判断能力
3. 极端功利婚恋观：明确将婚姻等同于物质索取，主张“结婚就是要改善生活/捞好处，自己只享受不付出”
4. 严重情绪认知障碍：认为“矛盾中自己永远没错，必须对方无条件妥协认错”，无基本共情与反思能力
5. 极端反常识认知：持有明显违背基本常识、被极端叙事深度裹挟的观点（如“努力完全没用”“所有人都是坏的”）
6. 零反思能力：坚称自己毫无缺点，所有问题与矛盾都归咎于外部环境或他人
7. 消费观严重失衡：主张“月光理所应当，恋爱中对方必须无条件为自己花钱”，无任何风险与责任意识
8. 极端消极处事态度：对工作、生活持彻底怨天尤人态度，无任何主动解决问题的意愿

### 总分评级标准
- ≥85分：优秀适配。三观高度契合，逻辑清晰，边界感强，长期相处内耗极低、成长性高
- 60~84分：合格适配。无核心观念冲突，认知符合普遍基准，部分维度可进一步磨合
- ＜60分：不适配。存在多处认知断层与观念硬伤，长期相处内耗风险极高

## 三、断层线分析使用提示
1. **禁止当场打分**：避免主观第一印象偏差，建议对话结束后凭真实回答复盘记录，重点记录推导过程而非结论。
2. **交叉验证原则**：同一特质需通过2个以上不同问题的回答交叉印证，避免单次叙事伪装，出现逻辑矛盾即为断层点。
3. **优先级规则**：红线判定＞维度一致性＞总分；多个维度连续出现不合格的，即使总分过线也需谨慎。
4. **细节优先原则**：优先采信有具体事例、承认自身局限性的回答；全是标准化话术、绝对化表述的回答，标记为高叙事伪装风险。
