# 🛠 Life Tool

> 个人开发实用工具集合 —— 包含算法练习、DNS 延迟测试、私密聊天工具及 Windows 维护脚本。

## 📦 模块概览

| 模块                     | 说明                                                         |
| ------------------------ | ------------------------------------------------------------ |
| `algorithm_test_project` | Java Maven 项目，内含 200+ 算法题解（LeetCode / 剑指 Offer 风格） |
| `bc_large_dc`            | 大数据连接与处理测试项目（含多版本 Main 实现）               |
| `dns`                    | DNS / DoH 服务可用性与延迟测试工具（Java）                   |
| `secret_chat`            | 纯前端私密聊天工具（HTML，支持端到端加密思想）               |
| `windows`                | Windows 系统维护脚本（PowerShell，清理无效目录）             |
